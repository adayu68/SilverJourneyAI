三智能体协作推荐

健康 Agent · 安全 Agent · 兴趣 Agent 三维度独立评分，Moderator 裁判器加权融合，安全分低于 2 星自动过滤。

多方言语音交互

Whisper 支持普通话、粤语、四川话、闽南话，老人直接说话即可操作，无需打字。

🛡️

实时安全守护

综合评估医疗资源距离、实时天气风险、景区人流密度，高风险目的地一律过滤不推荐。

📱

子女联动小程序

微信小程序实时 GPS 位置同步，老人超出安全区域自动推送预警给子女，一键紧急呼叫。

自动生成大字版行程图片、慢速语音导览音频、紧急联系卡，出发前准备零压力。

快速开始

3 步启动，5 分钟体验完整功能

1

克隆项目

git clone https://github.com/adayu68/SilverJourneyAI.git
cd SilverJourneyAI
2

安装依赖

pip install -r requirements.txt
3

启动应用

# 老人端 Web 界面
streamlit run app.py

# 子女小程序后端 API
python api_server.py

浏览器打开 http://localhost:8501 即可体验
